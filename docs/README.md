# Geometric lib

**Проект создан для нахождения площади и периметра геометрических фигур**

## Математические формулы

### Площади:
- Окружность: S = πR²
- Треугольник: S = ah/2
- Прямоугольник: S = ab
- Квадрат: S = a²

### Периметры:
- Окружность: P = 2πR
- Треугольник: P = a + b + c
- Прямоугольник: P = 2a + 2b
- Квадрат: P = 4a

## Функции для фигур:

### Окружность (`circle.py`)
- **Нахождение площади окружности по радиусу:** **`area(r)`**  
    **Пример:** `area(10) -> 314.1592653589793`  
- **Нахождение длины окружности по радусу:** **`perimeter(r)`**  
    **Пример:** `perimeter(10) -> 62.83185307179586`

### Треугольник (`triangle.py`)
- **Нахождение площади треугольника по основанию и проведенной к нему высоте:** **`area(a, h)`**  
    **Пример:** `area(10, 5) -> 25`
- **Нахождение периметра треугольника по трём сторонам:** **`perimeter(a, b, c)`**  
    **Пример:** `perimeter(10, 5, 1) -> 16`
    
### Прямоугольник (`rectangle.py`)
- **Нахождение площади прямоугольника по двум сторонам:** **`area(a, b)`**  
    **Пример:** `area(10, 5) -> 50`
- **Нахождение периметра прямоугольника по двум сторонам:** **`perimeter(a, b)`**  
    **Пример:** `perimeter(10, 5) -> 30`
    
### Квадрат (`square.py`)
- **Нахождение площади квадрата по стороне:** **`area(a)`**  
    **Пример:** `area(10) -> 100`

- **Нахождение периметра квадрата по стороне:** **`perimeter(a)`**  
    **Пример:** `perimeter(10) -> 40`
